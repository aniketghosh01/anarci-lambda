# ngs-pipeline/v2.0.0

NGS-pipeline meta data fields:

* ngs_run: Benchling NGS RUN
* ngs_pp: Benchling NGS PP
* ngs_type: Benchling NGS type (Clone/Clone Pool)
* project: Benchling project
* entry: Benchling notebook entry
* software: NGS-pipeline version
* sequencer: Equipment details (optional)
* user: User details
* time: Creation time
* kumo_files: List of NGS RUN files in Kumo
* related_files: List of NGS RUN related files in TDP
